# 12 — 配色优化：球场地胶设计系统

> 背景：当前设计系统以大面积白底 + 浅色文字为主，远距离/视力不佳时看起来"一片白"。
> 经过讨论，确定以**真实羽毛球场地胶**为视觉核心，融合 F1（球场色 Hero）+ F4（球场结构布局）方案，
> 同时修复 WCAG 对比度不达标的问题。
>
> Demo 文件：`docs/demo-v2-court-accurate.html`（浏览器打开可交互预览）

---

## 一、确认的设计方向

### 1.1 核心概念

**整个页面就是一张球场地胶垫的俯视图。**

```
┌─────── 外围深色 surround ────────┐
│  导航栏（白字）                    │  ← 球场垫边缘
│  ┌─── 内场 court 色 ───────────┐  │
│  │  Hero · 白色文字 · 球场 SVG  │  │  ← 球场上半场
│  │  ═══════ 球网分隔线 ════════  │  │  ← 中线 = 分隔
│  │  ┌── 白底内容区 ──────────┐  │  │
│  │  │ 知识库 Bento 卡片      │  │  │  ← 白色区域
│  │  │── 球场线条分隔 ────────│  │  │  ← 绿色细条
│  │  │ 教学日记（微旋转卡片） │  │  │  ← 暖白底
│  │  └────────────────────────┘  │  │
│  │  CTA 区域（球场色底）        │  │  ← 回到球场色
│  └──────────────────────────────┘  │
│  页脚（白字）                      │  ← 球场垫边缘
└────────────────────────────────────┘
```

### 1.2 确认的选项

| 编号 | 决策项 | 结论 |
|------|--------|------|
| A | 辅助文字对比度 | `#9CA3AF` → `#6B7280`，达标 WCAG AA |
| C | 边框/分隔线 | `#F3F4F6` → `#E5E7EB`，加深一档 |
| F1 | 首页 Hero | 球场色背景 + 白色文字 |
| F4 | 页面框架 | body = surround 深色，内容 = court 色框 |
| F1+F4 | 融合 | Hero 球场色 + 外围框架 + 球网分隔线 + 白色内容区 |
| 知识库文章 | 分类色系 | 手法=绿、步法=蓝、运动科学=橙红，Header 用球场色 |
| 教学日记 | 独立风格 | 每篇独立设计，不受球场色系约束 |
| 球场配色预设 | 6 套主题 | 经典绿 / 蓝绿 / 红蓝 / 深海蓝 / 绿橙 / 灰蓝 |

## 三、BWF 标准球场 SVG

### 3.1 尺寸参数

```
全场长度:     13.40m
双打宽度:     6.10m
单打宽度:     5.18m（距双打边线 0.46m 每侧）
球网位置:     6.70m（正中间）
前发球线:     距球网 1.98m（y = 4.72m / 8.68m）
双打后发球线: 距底线 0.76m（y = 0.76m / 12.64m）
中线:         x = 3.05m，仅存在于发球区（前发球线→底线）
线宽:         40mm
```

### 3.2 SVG viewBox 坐标映射

比例尺：1m = 40px → viewBox `0 0 244 536`

```
Y 坐标:
  y=0      底线（上）
  y=30.4   双打后发球线（上）
  y=188.8  前发球线（上）
  y=268    球网
  y=347.2  前发球线（下）
  y=505.6  双打后发球线（下）
  y=536    底线（下）

X 坐标:
  x=0      双打左边线
  x=18.4   单打左边线
  x=122    中线
  x=225.6  单打右边线
  x=244    双打右边线
```

### 3.3 公共 SVG 组件

球场 SVG 在多处复用（Hero 水印、文章 Header 装饰），建议抽取为 Astro 组件：

**文件：`src/components/common/CourtSVG.astro`**

```astro
---
interface Props {
  class?: string;
  strokeColor?: string;
  strokeWidth?: number;
}
const {
  class: className = '',
  strokeColor = 'white',
  strokeWidth = 2.5,
} = Astro.props;
const sw = strokeWidth;     // 外框线宽
const swi = sw * 0.6;       // 内线线宽
---
<svg class={className} viewBox="-2 -2 248 540" fill="none" xmlns="http://www.w3.org/2000/svg">
  <!-- 外框（双打边界） -->
  <rect x="0" y="0" width="244" height="536" stroke={strokeColor} stroke-width={sw} rx="0.5"/>
  <!-- 单打边线 -->
  <line x1="18.4" y1="0" x2="18.4" y2="536" stroke={strokeColor} stroke-width={swi}/>
  <line x1="225.6" y1="0" x2="225.6" y2="536" stroke={strokeColor} stroke-width={swi}/>
  <!-- 球网 -->
  <line x1="0" y1="268" x2="244" y2="268" stroke={strokeColor} stroke-width={sw}/>
  <!-- 前发球线 -->
  <line x1="18.4" y1="188.8" x2="225.6" y2="188.8" stroke={strokeColor} stroke-width={swi}/>
  <line x1="18.4" y1="347.2" x2="225.6" y2="347.2" stroke={strokeColor} stroke-width={swi}/>
  <!-- 双打后发球线 -->
  <line x1="18.4" y1="30.4" x2="225.6" y2="30.4" stroke={strokeColor} stroke-width={swi}/>
  <line x1="18.4" y1="505.6" x2="225.6" y2="505.6" stroke={strokeColor} stroke-width={swi}/>
  <!-- 中线（仅发球区内） -->
  <line x1="122" y1="0" x2="122" y2="188.8" stroke={strokeColor} stroke-width={swi}/>
  <line x1="122" y1="347.2" x2="122" y2="536" stroke={strokeColor} stroke-width={swi}/>
</svg>
```

## 五、不做的事情

- **不做暗色模式** — 球场色系本身已经提供了深色区域（Hero、nav、footer），不需要全站暗色
- **不改教学日记风格** — 每篇独立设计，不受球场色系约束
- **不做 F4 全页球场布局** — 留作未来特殊页面（关于页、404 页）的创意方案
- **不做首期主题切换** — MVP 先用经典绿，主题切换功能后续迭代
