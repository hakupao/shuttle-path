## 二、球场线条规格

| 线条 | 实现方式 | 粗细 | 颜色 |
|------|---------|------|------|
| 外框（双打边线） | `.court` 的 `border` | 4px | 纯白 `white` |
| 单打边线（左） | `.court::after` | 4px | 纯白，距左 7.5% |
| 单打边线（右） | `.singles-line-right` | 4px | 纯白，距右 7.5% |
| 后发球线（上） | `.court-row-back-service` 的 `border-bottom` | 4px | 纯白 |
| 前发球线 | `.court-line-h` | 4px | 纯白，`margin: 0 7.5%` |
| 中线 | `.court-half-left` 的 `border-right` | 4px | 纯白 |
| 中线渐隐（Hero） | `.hero-center-line` | 4px | 白→透明渐变 |
| 球网 | `.court-net` | 4px | 纯白 |
| 后发球线（下） | `.court-row-footer` 的 `border-top` | 4px | 纯白 |

**所有线条统一 4px 纯白，无圆点装饰。**

## 三、内容区域 padding

球场单打边线在 7.5% 处，内容需要在线内侧留白：

| 区域类型 | padding 规则 | 说明 |
|---------|-------------|------|
| 全宽区域（Hero/涨球记录/知识库） | `12%` 左右 | 在单打边线内侧，两侧留白均匀 |
| 左半场（手法/运动科学） | 左 `19%`，右 `8px` | 左边对齐单打边线内侧，右边紧贴中线 |
| 右半场（步法/© SP） | 左 `8px`，右 `19%` | 左边紧贴中线，右边对齐单打边线内侧 |

## 四、响应式策略

**移动端和桌面端显示完全一致**——所有球场线条齐全，布局不变。
仅调整 padding 数值和 min-height 让内容在小屏上更紧凑：

| 属性 | 移动端 | 桌面端 |
|------|--------|--------|
| 球场最大宽度 | 480px | 520px |
| Hero min-height | 280px | 360px |
| 手法步法 min-height | auto | 280px |
| 全宽 padding 上下 | 16px | 28px |
| 后发球线区高度 | 28px | 40px |

## 五、配色方案

全绿底白字，不再有白色区域：

```
球场底色:    --color-court     #1B8A4E
球场外围:    --color-court-surround  #147A3E（body 背景）
所有线条:    white（纯白）
所有文字:    white / white/80 / white/50 / white/35（不同层级）
```
