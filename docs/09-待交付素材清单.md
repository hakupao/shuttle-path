## 如何替换

拿到 SVG 文件后：

1. **图标类**（Logo、分类图标）：打开 SVG 文件，复制 `<svg>` 标签内的内容，粘贴到代码中对应位置替换占位 SVG。确保 stroke 使用 `currentColor`。
2. **插画类**（Hero、404）：放到 `public/images/illustrations/`，然后在代码中用 `<img src="/images/illustrations/xxx.svg" />` 引用，或内联 SVG
3. **装饰类**（羽毛碎片）：放到 `public/images/decorations/`，在 CSS 或 HTML 中引用

## 通用风格指引

**交付给画师/AI 时附上这段：**

> **Overall style guide for Shuttle Path icon set:**
>
> - Style: Hand-drawn ink sketch, like quick confident pen strokes on paper
> - Line quality: Natural thickness variation (thicker on downstrokes, thinner on curves), not perfectly uniform
> - Line gaps: Occasionally leave small gaps where lines meet — this creates the casual hand-drawn feel
> - Color: Use `currentColor` for stroke whenever possible, so CSS can control the color.
>   - On light backgrounds: dark warm gray `#3f3a36`
>   - On dark backgrounds (court green `#1B8A4E`, surround `#147A3E`): white `#FFFFFF`
> - Accent colors (when specified): green `#1B8A4E` and orange `#F59E0B`, used sparingly at low opacity (15-40%) as watercolor-style washes
> - Stroke: Round line caps (`stroke-linecap: round`), stroke width 1.5-3px depending on icon size
> - Format: SVG with transparent background, clean paths, no embedded raster images
> - Feel: Athletic but warm. Like a coach's whiteboard sketches — quick, clear, with personality. Not cute/kawaii, not rigid/corporate.
>
> **Important context:** The website uses a badminton court floor design system. The primary background color is court green (`#1B8A4E`) with white court line markings. Illustrations should feel at home on this "court surface" — like drawings on a coach's tactics board.
