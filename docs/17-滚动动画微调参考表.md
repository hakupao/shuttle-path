# 17 — 主页滚动动画微调参考表

> 基于 `src/pages/index.astro` 中 GSAP ScrollTrigger 动画 v3.1 的完整拆解。
> 用于微调时快速定位"第几圈滚轮→对应什么动画"。

---

## 1. 全局参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `.scene` 高度 | `700vh` | 滚动空间总高度 |
| 视口 (`.camera`) | `100vh` | 被 ScrollTrigger pin 固定 |
| **有效滚动距离** | **`600vh`** | 700vh - 100vh |
| `scrub` | `2` | 滚动到动画的缓冲延迟（秒） |
| `pinType` | `'transform'` | GPU 加速 pinning |
| Timeline 总时长 | **29.5 单位** | 8 个段落的 dur 之和 |

### 响应式参数

| 参数 | 移动端 (<768px) | 桌面端 (>=768px) | 含义 |
|------|----------------|-----------------|------|
| `P_ANGLE` | 18° | 22° | 全景俯角 |
| `P_SCALE` | 0.6 | 0.7 | 全景缩放 |
| `F_ANGLE` | 35° | 38° | 聚焦俯角 |
| `F_SCALE` | 2.6 | 2.2 | 聚焦缩放 |

## 2. 滚轮次数估算

标准鼠标滚轮一格 ≈ 100px。以下按不同视口高度估算：

| 设备 | 视口高度 | 滚动距离 (600vh) | 总滚轮次数 |
|------|---------|-----------------|-----------|
| MacBook Air 13" | ~750px | 4500px | **~45 次** |
| MacBook Pro 14" | ~850px | 5100px | **~51 次** |
| 外接 1080p 显示器 | ~920px | 5520px | **~55 次** |
| 外接 2K 显示器 | ~1200px | 7200px | **~72 次** |

> **以下所有计算以 MacBook Pro 14"（~51 次滚轮到底）为基准。**

## 3. 段落总览

Timeline 由 8 个段落顺序拼接：

| # | 类型 | 目标区域 | dur | 时间线范围 | 滚动占比 | 滚动距离 | 滚轮次数 |
|---|------|---------|-----|-----------|---------|---------|---------|
| 1 | **聚焦** | Hero 主视觉 | 5 | 0 → 5 | 16.9% | 101.7vh | **~9** |
| 2 | 过渡 | Hero → 涨球记录 | 2.5 | 5 → 7.5 | 8.5% | 50.8vh | **~4** |
| 3 | **聚焦** | 涨球记录 | 5 | 7.5 → 12.5 | 16.9% | 101.7vh | **~9** |
| 4 | 过渡 | 涨球记录 → 知识库 | 2.5 | 12.5 → 15 | 8.5% | 50.8vh | **~4** |
| 5 | **聚焦** | 知识库 | 5 | 15 → 20 | 16.9% | 101.7vh | **~9** |
| 6 | 过渡 | 知识库 → 分类 | 2.5 | 20 → 22.5 | 8.5% | 50.8vh | **~4** |
| 7 | **聚焦** | 手法/步法分类 | 5 | 22.5 → 27.5 | 16.9% | 101.7vh | **~9** |
| 8 | 退出 | 淡出收场 | 2 | 27.5 → 29.5 | 6.8% | 40.7vh | **~3** |
| | | | **29.5** | | **100%** | **600vh** | **~51** |

## 4. 逐次滚轮动画映射（基准: 51次）

下表以每次滚轮为单位，标注当前所在段落及视觉效果：

| 滚轮 # | 段落 | 段内进度 | 正在发生的动画 |
|--------|------|---------|--------------|
| **0** | — | 0% | 初始状态：全景俯角(22°)，球在 Hero 区，全部内容可见 |
| **1-3** | 聚焦 Hero | ~11-33% | 镜头下压(22°→38°)+放大(0.7→2.2)，暗角加深，聚光灯收缩亮起 |
| **4-5** | 聚焦 Hero | ~44-56% | 球降落到 Hero 中心(scale 1→0.75)，球光晕变暗(opacity 1→0.3)，非 Hero 内容淡出(→0.15) |
| **6-9** | 聚焦 Hero | ~67-100% | 镜头继续缓动到位(power3.inOut 后半段)，Hero 区完全聚焦 |
| **10** | 过渡 1 | ~25% | 镜头开始回拉(38°→22°)，球飞起(scale→1.3, rotation→+8°)，聚光灯扩散 |
| **11-12** | 过渡 1 | ~50-75% | 球到达 Hero↔涨球记录 中间位置，全景缩放恢复，暗角全开(opacity→1) |
| **13** | 过渡 1→聚焦涨球 | ~100%→0% | 过渡结束，开始聚焦涨球记录 |
| **14-16** | 聚焦涨球记录 | ~11-33% | 镜头再次下压+放大，球飞向涨球记录区(left→56%)，聚光灯跟随 |
| **17-18** | 聚焦涨球记录 | ~44-56% | 球降落，其他内容淡出，涨球记录区高亮(opacity→1) |
| **19-22** | 聚焦涨球记录 | ~67-100% | 镜头缓动到位，涨球记录完全聚焦 |
| **23** | 过渡 2 | ~25% | 镜头回拉，球飞起(rotation→-8°)，跨越球网 |
| **24-25** | 过渡 2 | ~50-75% | 球到达涨球记录↔知识库 中间，全景视角 |
| **26** | 过渡 2→聚焦知识库 | ~100%→0% | 过渡结束 |
| **27-29** | 聚焦知识库 | ~11-33% | 镜头下压+放大，球飞向知识库区(left→43%)，聚光灯跟随 |
| **30-31** | 聚焦知识库 | ~44-56% | 球降落，知识库区高亮 |
| **32-35** | 聚焦知识库 | ~67-100% | 镜头缓动到位 |
| **36** | 过渡 3 | ~25% | 镜头回拉，球飞起(rotation→+8°) |
| **37-38** | 过渡 3 | ~50-75% | 球到达知识库↔分类 中间 |
| **39** | 过渡 3→聚焦分类 | ~100%→0% | 过渡结束 |
| **40-42** | 聚焦分类 | ~11-33% | 镜头下压+放大，球飞向手法/步法区(left→50%) |
| **43-44** | 聚焦分类 | ~44-56% | 球降落，分类区高亮 |
| **45-48** | 聚焦分类 | ~67-100% | 镜头缓动到位 |
| **49** | 退出 | ~33% | 镜头倾斜(22°→14°)，球开始缩小淡出(scale→0.4, opacity→0) |
| **50** | 退出 | ~67% | 聚光灯消失，暗角消退，所有内容恢复(opacity→1) |
| **51** | 退出 | 100% | 动画结束，镜头最终状态(14°, scale 0.42)，球完全消失 |
