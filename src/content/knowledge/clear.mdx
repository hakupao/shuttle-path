高远球（Clear）是将球以高弧线打到对方底线附近的击球方式。它是后场技术的"母球"——吊球、杀球的动作框架都脱胎于高远球。

## 为什么高远球如此重要

- **防守利器**：被动时打出高远球，可以争取回中场的时间
- **技术基础**：高远球的挥拍动作是吊球、杀球的母体
- **发力教科书**：高远球的发力链最完整，练好它就理解了鞭打发力的精髓

## 发力链分析

高远球的力量来源于一条从下到上、从大关节到小关节的**动力链**：

```
蹬地 → 转髋 → 转肩 → 大臂外旋 → 前臂内旋 → 手腕闪动 → 手指收紧
```

每个环节就像鞭子的传导——下一个环节在前一个环节减速时加速，最终在击球点上达到速度峰值。这就是"鞭打发力"的核心原理。

**关键发力节点：**

| 环节 | 作用 | 常见错误 |
|------|------|---------|
| 蹬地转髋 | 提供最底层的力量来源 | 脚没有蹬地，纯靠手臂挥 |
| 转肩引拍 | 拉满身体的弹弓 | 侧身不够，面对球网挥拍 |
| 大臂带动 | 将身体力量传导到手臂 | 抬肘不够高，肘低于肩 |
| 前臂内旋 | 最关键的加速环节 | 只会"推"不会"旋" |
| 手腕闪动 | 最后的加速和方向控制 | 大幅甩手腕，导致受伤 |

## 击球点

正确的击球点在：

- **前后位置**：身体前方，约在持拍手同侧肩的正上方偏前
- **高度**：手臂完全伸展时的最高点
- **左右位置**：持拍手同侧，不要去打正头顶的球（那是头顶球的技术）

> 击球点偏后是初学者最常见的问题。当你觉得"怎么打都发不了力"时，八成是击球点太靠后了。

## 分解练习步骤

### 第一步：挥拍练习（不用球）

1. 侧身站立，非持拍手指向前方（指向来球方向）
2. 持拍手架拍到脑后，肘部与肩同高
3. 蹬地 → 转髋 → 转肩 → 挥拍，感受力量从脚传到手
4. 每组 20 次，每天 3 组

### 第二步：定点击球

1. 搭档在前场高手抛球到你的正手后场
2. 侧身、架拍、等球落到击球点
3. 不追求力量，只追求动作和击球点的正确
4. 能稳定打到对方底线后，逐渐加大力量

### 第三步：移动中击球

1. 中场准备 → 后退到位 → 侧身击球 → 回到中场
2. 重点练习后退步法和击球的衔接
3. 注意击球后的身体平衡和快速回位

## 常见问题

**球打不高、打不远？**

多数原因是击球点偏后或没有利用蹬地转体的力量。试试只用七分力，专注于击球点和发力顺序，往往效果比"使劲打"好。

**击球时手腕疼？**

手腕不应该承受主要发力。检查你的发力链——如果跳过了蹬地、转髋、转肩的环节，所有力量就压到了手腕上。要"用身体打球，不是用手打球"。
